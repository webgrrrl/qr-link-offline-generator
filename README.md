# QR Code Generator Extension

This is a Brave browser extension that generates a QR code image of the URL of the currently open tab and saves it to your Brave browser's default download location.

## Features

- Generates QR code for the active tab's URL
- Works offline
- Saves QR code as PNG to downloads

## Installation

1. Download the ZIP file of this repo, then unzip it so you have the extension folder (the folder must contain manifest.json).
1. Open Brave browser
1. Go to chrome://extensions/
1. Enable "Developer mode"
1. Click "Load unpacked"
1. Select this folder

## Usage

1. Click the extension icon
2. Click "Generate QR Code"
3. The QR code image will be downloaded
